## 🚀 Quick Start

### Prerequisites

```bash
# Verify prerequisites
node --version  # >= 18.0.0
npm --version   # >= 8.0.0
git --version   # >= 2.0.0
```

### Installation

```bash
# Clone the repository
git clone https://github.com/tiaastor/tiation-terminal-workflows.git
cd tiation-terminal-workflows

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
